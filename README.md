# Notes App Using MVVM
### A simple notes making android application that is made on MVVM pattern. 

## Tech Stack 
<ul>
<li>Kotlin</li>
<li>Room Database(for CRUD)</li>
</ul>

## Library Used - 
<ul>
<li>Intuit/SDP</li>
</ul>
